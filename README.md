# Installation Guide
Follow these steps exactly in this order. If you miss a step, Skype might update itself and you'll have to start over!
# 1. Download the Files
Download the Skype .exe installer provided in this repository and the SkypeFixer.bat script.
# 2. Go Offline
Disconnect your Wi-Fi or unplug your Ethernet cable. This is the most important step. If Skype has internet during the install, it will immediately force an update.
# 3. Install Skype
Run the skype_5.7.0.123.exe (or your chosen version) while offline. Follow the prompts to finish the installation.
# 4. Close Skype Immediately
If Skype opens automatically after the installer finishes:
Press Ctrl + Shift + Esc to open Task Manager.
Find Skype in the list.
Right-click it and select End Task.
# 5. Run the SkypeFixer
Locate SkypeFixer.bat in your folder.
Right-click the file and select Run as Administrator.
A black window will appear. Once it says "Done!", you can close it.
